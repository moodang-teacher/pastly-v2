Chapter 1

🏫 기수 생성

기수는 학생을 묶는 반(班) 단위입니다. 학생을 배정하기 전에 기수를 먼저 만들어 주세요.

만드는 방법

  1. 관리자 페이지 → 기수 관리 탭
  2. 기수명 입력 (예: 2025-1기, 7월반)
  3. 생성 버튼 클릭 → "활성 기수" 목록에 추가됨

아카이브 (수료 처리)

수업이 끝난 기수는 아카이브 버튼으로 보관 처리합니다.

  • 학생 기록은 그대로 남고, 신규 배정만 막힘
  • 아카이브된 기수는 하단 "아카이브" 섹션으로 이동
💡 기수가 하나도 없으면 학생 배정을 할 수 없습니다. 학생 등록 전에 기수를 먼저 만드세요.
Chapter 2

👩‍🎓 학생 관리

학생 가입 흐름

  1. 학생이 앱에서 회원가입 → 전공과 선생님 선택
  2. 가입 직후 기수 배정 대기 상태로 목록에 나타남
  3. 선생님이 배정할 기수를 선택 후 배정
  4. 배정 완료 → 학생이 시험을 볼 수 있게 됨

배정 대기 목록

관리자 페이지 → 학생 관리의 상단 섹션에 표시됩니다.

버튼설명
배정선택한 학생을 기수에 개별 배정
선택 N명 배정체크박스로 고른 학생을 한 번에 배정
전체 배정대기 중인 모든 학생을 같은 기수에 배정
전공변경다른 선생님 반으로 이동 (기수 배정 초기화)

배정된 학생 목록

이미 기수에 배정된 학생 전체를 이름 검색으로 찾을 수 있습니다.

항목내용
레벨풀어본 문제 수 기반 자동 산정
최고점수지금까지의 최고 모의고사 점수
문제 수누적 풀이 문제 수 (= XP)
❌ 버튼기수 배정 해제 (학생 계정 삭제 아님)
ℹ️ 기수 배정을 해제하면 해당 학생은 다시 "배정 대기" 상태로 돌아갑니다.

Chapter 5

🤖 Google Gems로 JSON 만들기

📎 Gems 바로가기
  1. 위 링크에서 Gems 접속 (Google 로그인 필요)
  2. 기출문제 PDF 또는 이미지를 채팅창에 업로드
  3. 기출문제용 또는 모의고사용 중 선택해서 요청
  4. Gems 화면 상단의 다운로드 버튼으로 .json 파일 저장
💡 메모장에 복사/붙여넣기도 가능하나, 실수를 줄이기 위해 가급적 다운로드를 이용하세요.
Chapter 6

⬆️ 업로드 방법

  1. 관리자 페이지 → 문제 관리
  2. 전공 / 시험 유형 선택
  3. 미용 공통 과목이면 "공통 과목" 토글 ON
  4. JSON 파일 선택 → 문제 수 확인 → 업로드

JSON 형식 참고

answer_index: 1번 정답=0, 2번=1, 3번=2, 4번=3

{
  "questions": [
    {
      "category": "헤어미용",
      "question_text": "문제 내용",
      "options": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer_index": 2,
      "explanation": "해설 (생략 가능)"
    }
  ]
}
Chapter 7

📤 문제 내보내기 (다른 선생님과 공유)

등록된 문제 전체를 JSON 파일로 내보내서 다른 전공 선생님에게 공유할 수 있습니다.

내보내기 방법

  1. 관리자 페이지 → 문제 관리 탭
  2. 아래쪽 "등록된 전체 문제" 섹션 우측의 JSON 내보내기 클릭
  3. 전공명_날짜.json 파일이 자동 다운로드됨
  4. 파일을 카카오톡·메일 등으로 상대 선생님에게 전달

받은 파일 올리기

  1. 받은 JSON 파일을 Chapter 6 업로드 방법 순서대로 올리면 됨
  2. 기출/모의고사 구분이 파일에 포함되어 있어 시험 유형을 따로 선택하지 않아도 자동 반영됨
💡 같은 시험을 준비하는 두 전공이 있다면, 한 쪽에서 내보내서 파일을 공유하면 문제 풀을 똑같이 맞출 수 있습니다.
Chapter 8

🔗 공통 과목 안내

공중위생관리학 · 화장품학 · 피부학은 모든 미용 세부전공 학생이 공유합니다.

  • 헤어 선생님이 올린 공통 문제 → 네일·피부·메이크업 학생도 풀게 됨
  • 중복 업로드 OK — 시스템이 자동으로 섞어서 출제
