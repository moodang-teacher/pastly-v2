# 📝 선생님용 문제 업로드 가이드

## 1. AI로 JSON 파일 만드는 방법

### 기본 형식

아래 형식으로 JSON 파일을 만들어야 합니다:

```json
{
  "questions": [
    {
      "category": "헤어미용",
      "question_text": "문제 내용을 입력하세요",
      "options": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer_index": 2,
      "explanation": "해설을 입력하세요 (없어도 됩니다)"
    }
  ]
}
```

### answer_index 규칙
- 1번이 정답 → `0`
- 2번이 정답 → `1`  
- 3번이 정답 → `2`  
- 4번이 정답 → `3`

## 2. Claude (클로드)로 만드는 방법

### 단계 1: Claude.ai 접속
https://claude.ai 에 접속하여 로그인합니다.

### 단계 2: PDF 또는 이미지 업로드
문제가 있는 PDF 파일이나 캡쳐 이미지를 채팅창에 드래그해서 올립니다.

### 단계 3: 아래 프롬프트 복사해서 붙여넣기

```
첨부한 파일에서 문제들을 추출해서 아래 JSON 형식으로 만들어줘.
answer_index는 0부터 시작해 (1번=0, 2번=1, 3번=2, 4번=3).

{
  "questions": [
    {
      "category": "과목명",
      "question_text": "문제 내용",
      "options": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer_index": 정답번호,
      "explanation": "해설"
    }
  ]
}

category는 다음 중 하나로 통일해줘:
- 헤어미용 (헤어 관련)
- 공중위생관리학 (위생 법규 관련)
- 피부학 (피부 구조, 피부 타입 등)
- 화장품학 (화장품 성분, 종류 등)

다른 형식 없이 JSON만 출력해줘.
```

### 단계 4: 파일로 저장
Claude가 출력한 내용을 전체 복사 → 메모장에 붙여넣기 → `헤어_기출_20250101.json` 같은 이름으로 저장

---

## 3. 업로드 방법

1. 관리자 페이지 접속 → **문제 관리** 탭
2. **전공/과목** 선택 (본인 전공)
3. **시험 유형** 선택:
   - 📚 기출문제 = 실제 자격증 기출
   - ✏️ 모의고사 = 학교 수업용 시험
   - ⚡ Crash Test = 랜덤 출제용
4. 미용 공통 과목이면 **"공통 과목"** 토글 켜기
5. JSON 파일 선택
6. 문제 수 확인 후 **업로드** 클릭

---

## 4. 자주 하는 실수

| 실수 | 해결책 |
|------|--------|
| 파일 오류 난다 | Claude에게 "JSON만 출력해줘, 다른 설명 없이"라고 다시 요청 |
| 문제 수가 안 맞는다 | 같은 파일을 나눠서 여러 번 올려도 됨 (자동으로 합쳐짐) |
| 그림이 있는 문제 | 이미지 문제는 현재 직접 입력 필요 (개발팀 문의) |

---

## 5. 공통 과목 올릴 때

미용 공통 과목(공중위생관리학, 화장품학, 피부학)은 **모든 세부전공 학생에게 공유**됩니다.
- 헤어 선생님이 올린 공통 문제 → 네일 학생도 풀게 됨
- 중복 업로드 괜찮음 (시스템이 알아서 섞어서 냄)
- 많이 올릴수록 좋음! 📚
